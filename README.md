
## Week 1 — Python + Math + Statistics for ML
Python fundamentals, linear algebra, calculus, and core statistics for machine learning.

## Week 2 — Preprocessing, EDA + Feature Engineering & Leakage
Data cleaning, exploratory data analysis, feature engineering techniques, and preventing data leakage.

## Week 3 — Regression + Bias-Variance & Cross-Validation
Linear and polynomial regression, evaluating the bias-variance tradeoff, and robust cross-validation strategies.

## Week 4 — Classification + KNN, SVM & Imbalanced Data
Classification metrics, K-Nearest Neighbors, Support Vector Machines, and handling imbalanced datasets.

## Week 5 — Trees & Ensembles + Tuning & Pipelines
Decision trees, random forests, gradient boosting, hyperparameter tuning, and Scikit-Learn pipelines.

## Week 6 — Unsupervised Learning + Anomaly Detection & Recommenders
Clustering algorithms, dimensionality reduction, anomaly detection, and recommendation systems.

## Week 7 — Deep Learning + Computer Vision + Transfer Learning
Neural network architectures, convolutional neural networks (CNNs), and transfer learning for computer vision.

## Week 8 — NLP + Transformers + LLMs, Embeddings & RAG
Natural language processing, transformer architectures, vector embeddings, Large Language Models, and RAG pipelines.

## Week 9 — Deployment + MLOps: Tracking, Serving, Monitoring
Experiment tracking, model serving APIs, CI/CD for machine learning, and production monitoring.

## Week 10 — Interpretability, Responsible AI + Capstone
Model explainability (SHAP/LIME), bias and fairness in AI, and the comprehensive capstone project.
